Find the Minions

Project information

  • Category: JavaScript
  • Project date: 21 November 2019

A wack-a-mole style game to find all the minions before the dreaded Yeti makes an appearance.Once the Yeti is found, the game reset and be ready for another round. includes:a score box,a reset button,randomize the placement of the minions/yeti,sound effects to each click.
